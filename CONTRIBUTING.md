# Contributing to EligiusHiddenArmor

First off, thank you for considering contributing to EligiusHiddenArmor! It's people like you that make it such a great plugin.

### Code Style
- Use Java 21 features where appropriate.
- Follow standard Java naming conventions.
- Make sure all methods have descriptive names.

### Development Process
1. Fork the repository.
2. Create a new branch for your feature or bug fix.
3. Make your changes.
4. Ensure the plugin compiles using `./gradlew clean build shadowJar`.
5. Submit a pull request.

### Reporting Bugs
If you find a bug, please create an issue with as much detail as possible, including:
- Minecraft Server version (Paper, Spigot, Folia)
- Plugin version
- Stack trace (if applicable)
- Steps to reproduce the bug
