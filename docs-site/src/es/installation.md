# Instalación

## Requisitos del Sistema

- **Java**: Java 21 o superior.
- **Versiones de Minecraft**: 1.21 hasta 1.21.x
- **Software Soportado**: Spigot, Paper, Purpur, Folia y otros forks.

## Dependencias

**EligiusHiddenArmor** requiere que instales **PacketEvents** en tu servidor. Debes descargarlo y colocarlo en tu carpeta de plugins.

## Pasos de Instalación

1. Descarga e instala [PacketEvents](https://modrinth.com/plugin/packetevents) en la carpeta `plugins` de tu servidor.
2. Descarga el archivo `EligiusHiddenArmor-1.0.1.jar` más reciente de la página oficial.
3. Coloca el archivo `.jar` dentro de la carpeta `plugins` de tu servidor.
4. Inicia el servidor para generar los archivos de configuración por defecto.
5. (Opcional) Configura `config.yml` con tus credenciales de base de datos y Redis si usas una red proxy.
6. Reinicia el servidor.
