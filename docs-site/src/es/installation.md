# Instalación

## Requisitos del Sistema

- **Java**: Java 21 o superior.
- **Versiones de Minecraft**: 1.21 hasta 26.1.2+
- **Software Soportado**: Spigot, Paper, Purpur, Folia y otros forks.

## Dependencias

**EligiusHiddenArmor** es completamente independiente. Incluye PacketEvents de forma nativa, lo que significa que no necesitas instalar ningún plugin de API externa.

## Pasos de Instalación

1. Descarga el archivo `EligiusHiddenArmor-1.0.0.jar` más reciente de la página oficial.
2. Coloca el archivo `.jar` dentro de la carpeta `plugins` de tu servidor.
3. Inicia el servidor para generar los archivos de configuración por defecto.
4. (Opcional) Configura `config.yml` con tus credenciales de base de datos y Redis si usas una red proxy.
5. Reinicia el servidor.
