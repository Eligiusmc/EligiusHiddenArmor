# HiddenArmor Wiki

Bienvenido a la wiki de HiddenArmor. Este plugin permite a los jugadores ocultar su armadura para mostrar sus skins, utilizando PacketEvents para asegurar que la armadura siga funcionando mecánicamente sin que se vea y respetando el idioma nativo del cliente para cada ítem.

## Características
- Soporte para Folia y Paper 1.21+.
- Integración asíncrona mediante PacketEvents.
- Traducción e internacionalización nativa delegada al cliente usando componentes (TranslatableComponents).
- Soporte para ignorar ciertas piezas de armadura.
- Opción de auto-ocultarse con pociones de invisibilidad.
