# Commands and Permissions

| Comando | Permiso | Descripción |
| --- | --- | --- |
| `/togglearmor` | `hiddenarmor.toggle` | Oculta/Muestra tu armadura. |
| `/hiddenarmor <jugador>` | `hiddenarmor.toggle.other` | Oculta/Muestra armadura de otro. |
| `/hiddenarmor reload` | `hiddenarmor.reload` | Recarga la configuración. |
